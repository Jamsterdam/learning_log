# 报价系统 
Github上加一个项目
做一个报价及价格变动追溯系统，可提醒需调价项目，可生成不同报表；可生成不同的监控仪表盘。有表单录入功能。有vimium快捷键功能。

- 录入表单：型号1，配置，价格，档线（自动计算跳出）
型号2 ……

- 客户名称（链接客户数据库）

- 可对比其他客户历史报价；
- 可自动生成排箱信息，（链接排箱信息）
- 手动录入海运，保险，检测等费用。

- 可生成报表：
Quotation, PI
各客户报价记录；
 
- 附加功能：一个客户可对接多个销售，助理，工程师，一个销售报价，另一个销售以及助理将会收到提醒。

全局要求：
- 解决高并发情况（全球最大销售团队多少人？）
- 各数据库应该和报价系统相互独立，可切分，可独立变动生长。
